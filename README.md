# blog

Personal blog of Eran Pinhas. Plain static HTML pages with one global stylesheet — no JavaScript, no build step.

- `index.html` — post list
- `posts/` — one HTML file per post
- `styles.css` — global stylesheet (light/dark via `prefers-color-scheme`)
- `rss.xml` — feed (hand-maintained)

Design source: [Claude Design project](https://claude.ai/design/p/f73bc3c3-e6d2-4f66-9059-fc512bc321c3).
